«От простого к сложному» - а ведь все действительно просто!!!!

Это ведь обычные операторы 
+ прибавит к числу один
- отнимет от числа один
~ логический NOT
Если оператор предназначен для двух переменных такой как
| - логический OR
& - логический AND
^ - логический XOR
То за ним в скобках следует закодированная вторая переменная. Закодирована она в виде звездочек - «*» и «$». Причем первые идут десятки (если они есть) — и они обозначаются «*», а через пробел единицы - «$». Например число 25 - ** $$$$$, 12 - * $$, 70 - *******.
Порядок следования операторов слева направо.

Все что написано на чаше строка за строкой было скопировано в файл cipher.txt

Для вычисления первого числа для первой строки  возьмите для старта -  число равное номеру задачи, для которой сундуки , не открыть (тут пригодиться результат первой задачи). Результат первой строки (он же первое нужное нам число - подсказка оно равно 4), является стартовым значением для второй.  Результат второй строки, стартовое для третьей и т. д.
